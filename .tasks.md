some notes during a kata

within a lambda, the return value is the value of the last expression evaluated. For example:

```kotlin
val sum = { a: Int, b: Int ->
  println("Calculating sum of $a and $b")

  a + b
}
println(sum(3, 4))
// Output: Calculating sum of 3 and 4
// Output: 7
```

`sum` vs `sumOf` on lists:

- `sum` is a function that directly sums the elements of a collection.
  ```kotlin
  val numbers = listOf(1, 2, 3, 4)
  val total = numbers.sum() // total is 10
  ```
- `sumOf` is a more flexible function that allows you to specify a transformation before summing.
  ```kotlin
  val items = listOf("apple", "banana", "cherry")
  val totalLength = items.sumOf { it.length } // totalLength is 16
  ```
